# `dewi-kadita` Analysis Scripts

Supporting scripts for analyzing the four canonical fish schooling scenarios from [`dewi-kadita`](https://github.com/sandyherho/dewi-kadita).

## Scripts

| Script | Description |
|--------|-------------|
| `spatiotemporal_evolution.py` | 4Ã—3 multipanel 3D visualization of collective states |
| `order_parameters_evolution.py` | Time evolution of polarization (P) and rotation (M) |
| `entropy_metrics_evolution.py` | Time evolution of oceanic entropy metrics |
| `final_state_entropy_profile.py` | Final state entropy profile comparison |

## Usage

```bash
cd scripts
python spatiotemporal_evolution.py
```

Requires NetCDF data files in `../data/` (generated by `dewi-kadita` simulations).
